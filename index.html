Vim mastery

Copy (yank)
  • Copy one line with yy
  • Copy two lines with 2yy
  • Copy word under cursor with yw
Paste (put)
  • Paste with p
Delete (cut)
  • Delete one line with dd
  • Delete two lines with 2dd
  • Delete word under cursor with dw
Cursor movement
  • Jump to the start of a line with 0
  • Jump to the first character on a line with ^
  • Jump to the end of a line with $
  • Jump to the first line of the document with gg
  • Jump to the last line of the document with G
Save (write) & close
  • Close after making no changes with :q
  • Force close after making changes with :q!
  • Save with :w
  • Save and quit with :wq
  • Save and quit across all tabs with :wqa
Search & replace
  • Search for pattern with /pattern
  • Replace text with :%s/old/new/g
